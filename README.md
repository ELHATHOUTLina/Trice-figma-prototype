#TRICE — Prototype Figma (application mobile)

TRICE est une application mobile de transport/mobilité conçue pour centraliser plusieurs services : création de compte, gestion de carte de transport (commande/rechargement), réservation de vélos/trottinettes, consultation d’horaires/itinéraires et support utilisateur.




## Prototype
- Lien Figma : https://www.figma.com/design/6PmLzA6EyYVEIODsm5Depc/TRICE?node-id=0-1&t=Y6t0s4HuVWzlBLwC-1 




## Principales fonctionnalités :

### Côté utilisateur
- Créer un compte et se connecter (avec vérification d’identité via justificatifs). 
- Commander une carte (physique ou virtuelle) avec choix d’abonnement et réductions. 
- Recharger une carte existante (paiement CB/PayPal/ONEY selon le rapport). 
- Réserver un vélo/trottinette selon station/lieu et disponibilité. 
- Consulter itinéraires/horaires (intégration/redirection type IDF Mobilités). 
- Aide et support (bot + contact agent si nécessaire). 

### Côté administration / agents
- Gestion des comptes (ajout, modification, suppression, fusion). 
- Gestion des tarifs, statistiques/rapports, historiques, dédommagements.
- Impression de cartes et suivi des opérations. 
- Maintenance : signalement et suivi des incidents/interventions.
